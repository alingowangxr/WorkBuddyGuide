# 附錄 A 常用指令模板

## A.1 檔案整理

```text
請整理當前資料夾中的檔案。
按檔案型別和主題分類，生成新的資料夾結構。
在執行前先給我一個整理方案，不要直接移動檔案。
```

## A.2 Excel 分析

```text
請分析這個 Excel 檔案。
請輸出：核心指標、異常資料、趨勢變化、可能原因、建議行動。
請生成一個摘要報告和圖表。
```

## A.3 PPT 生成

```text
請根據這份 Word 文件生成一份 PPT 大綱。
要求：10 頁左右，每頁包含標題、3-4 個要點和建議圖表。
風格正式，適合管理層彙報。
```

## A.4 會議紀要

```text
請整理這段會議內容。
輸出：會議結論、待辦事項、負責人、截止時間、風險點、未決問題。
```

## A.5 行業調研

```text
請調研【行業/公司/產品】。
輸出：市場背景、主要玩家、競品對比、趨勢判斷、機會點、風險點。
請附上資訊來源連結。
```

## A.6 銷售方案

```text
請基於客戶資料生成一份售前方案。
包括：客戶背景、痛點分析、推薦場景、實施路徑、預期收益、演示流程。
```
