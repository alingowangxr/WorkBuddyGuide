# WorkBuddy 閱讀與學習指南：從入門到團隊落地

這是一條面向新手、任務實踐者和團隊負責人的 WorkBuddy 學習路線。你不必從頭背功能：先找到當前最接近的工作問題，完成一項真實任務，再沿著使用手冊、實戰案例、進階系統和團隊落地逐步深入。

## WorkBuddy 新手入門路線

從[第一篇：先把 WorkBuddy 用起來](/bluebook/%E7%AC%AC%E4%B8%80%E7%AF%87%20%E4%BD%BF%E7%94%A8%E6%89%8B%E5%86%8A%EF%BC%9A%E5%85%88%E6%8A%8A%20WorkBuddy%20%E7%94%A8%E8%B5%B7%E4%BE%86/)開始，依次完成：

1. 下載、安裝和登入。
2. 認識主介面、任務與工作區。
3. 完成第一個可檢查的任務。
4. 載入一個真正用得上的 Skill。
5. 再學習聯結器、外部 API 與自動化任務。

建議邊讀邊做，不要只瀏覽截圖。每完成一章，至少留下一個可複用的提示詞、模板或操作記錄。

## 按真實任務選擇 WorkBuddy 教程

直接進入[第二篇：案例篇](/bluebook/%E7%AC%AC%E4%BA%8C%E7%AF%87%20%E6%A1%88%E4%BE%8B%E7%AF%87%EF%BC%9A%E5%BE%9E%E4%B8%80%E9%A0%85%E4%BB%BB%E5%8B%99%E5%88%B0%E4%B8%80%E6%94%AF%20AI%20%E5%9C%98%E9%9A%8A/)，按問題選擇章節：

- 處理 Word、Excel、PPT：第 11 章。
- 整理電腦檔案：第 12 章。
- 遠端控制電腦：第 13 章。
- 資訊、收藏、會議和投資：第 15～18 章。
- AI 影片、自媒體和 GEO：第 19～21 章。

案例跑通後，再回到第三篇，把你的做法整理為 Skill、自動化或多 Agent 系統。

## WorkBuddy 團隊落地路線

重點閱讀[第三篇：進階篇](/bluebook/%E7%AC%AC%E4%B8%89%E7%AF%87%20%E9%80%B2%E9%9A%8E%E7%AF%87%EF%BC%9A%E6%8A%8A%E6%A1%88%E4%BE%8B%E8%AE%8A%E6%88%90%E8%87%AA%E5%B7%B1%E7%9A%84%E5%B7%A5%E4%BD%9C%E7%B3%BB%E7%B5%B1/)與[第四篇：崗位與行業落地](/bluebook/%E7%AC%AC%E5%9B%9B%E7%AF%87%20%E5%B4%97%E4%BD%8D%E8%88%87%E8%A1%8C%E6%A5%AD%E8%90%BD%E5%9C%B0/)。

團隊實踐建議同時記錄：

- 輸入資料和許可權邊界。
- 任務拆解方式和失敗回退。
- 交付物的驗收標準。
- 可以複用的 Skill、模板和自動化。
- 哪些步驟必須由人確認。
